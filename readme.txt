Сборка с помощью makefile, лежащего в корневой папке проекта. Запуск из папки bin.

Клавиши управления:
z/x - смена времени суток (увеличить/уменьшить)
c/v - управление фонарем (увеличить/уменьшить яркость)
b - включить/выключить случайный ветер
n - включить/выключить смену времени суток
-/= - увеличить/уменьшить силу ветра (так же меняется направление ветра)
w/s, стрелочки - стандартные функции управления положением и направлением камеры
i/o - запись/чтение бинарного файла (bin/binary)
k/l - запись/чтение текстового файла (bin/text)

Использовались следующие библиотеки, устанавливаемые через apt-get:
libglew-dev
freeglut3-dev
libsoil-dev
libassimp-dev
